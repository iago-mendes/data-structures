## Application
> Create a Heaps implementation that takes 9 integers as input, creates a max-heap, and outputs a level-by-level representation.

- Input
	```
	3 25 1 100 36 19 2 17 7
	```
- Output
	```
	100
	19 36
	17 3 25 1
	2 7
	```
